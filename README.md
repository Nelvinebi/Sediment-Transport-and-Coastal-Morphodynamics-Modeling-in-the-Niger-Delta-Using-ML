Sediment Transport and Coastal Morphodynamics Modeling in the Niger Delta Using Machine Learning
📌 Project Overview

Coastal erosion and sediment redistribution are major environmental challenges in the Niger Delta, driven by complex interactions between waves, tides, currents, and sediment properties.

This project applies machine learning–based modeling to simulate sediment transport dynamics and coastal morphodynamic behavior using physically consistent synthetic oceanographic data. The outputs are designed to support coastal management, shoreline monitoring, and climate impact assessments, with GIS-ready spatial products.

🎯 Objectives

Simulate realistic coastal and oceanographic forcing conditions for the Niger Delta

Predict sediment transport rates using supervised machine learning

Assess coastal morphodynamics, including erosion and accretion tendencies

Generate GIS-compatible outputs for spatial coastal analysis and decision support

🗺️ Project Visuals / Outputs

Key outputs generated by this project include:

Erosion–Accretion Intensity Raster (GeoTIFF)
Spatial representation of sediment gain and loss intensity along the coastline

Classified Erosion and Accretion Zones (Shapefile)
Vector-based zoning suitable for coastal vulnerability mapping

All spatial outputs are compatible with QGIS and ArcGIS.

⚙️ Methodology

Synthetic Data Generation

Coastal forcing parameters generated using established sediment transport and coastal process relationships

Data Preprocessing

Feature scaling using Min–Max normalization

Data quality checks and validation

Machine Learning Modeling

Random Forest regression applied to predict sediment transport rates

Model Evaluation

Performance assessed using RMSE and R² metrics

Spatial Output Generation

Model predictions converted into GIS-compatible raster and vector layers

🤖 Models & Techniques

Algorithm: Random Forest Regressor

Why Random Forest?

Handles nonlinear coastal processes

Robust to multicollinearity

Provides interpretable feature importance

Additional Techniques:

Feature importance analysis

Spatial classification of morphodynamic zones

📈 Model Evaluation & Results

Evaluation Metrics:

Root Mean Square Error (RMSE)

Coefficient of Determination (R²)

Results Summary:

The model successfully captured relationships between hydrodynamic forcing and sediment transport behavior

Feature importance highlighted the dominant role of wave height, current velocity, and sediment grain size

🔍 Key Insights

Wave and current dynamics are primary drivers of sediment redistribution in the Niger Delta

Areas of persistent erosion and accretion can be spatially delineated using ML-derived indices

Machine learning provides a computationally efficient alternative for exploratory coastal morphodynamic assessments
